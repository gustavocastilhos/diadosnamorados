## 💡 Sobre o Projeto

Cansado dos clichês de Dia dos Namorados? Este projeto subverte as expectativas. Ao abrir o site, o usuário se depara com uma tela cinza e um contrato de **Termos e Condições (EULA)** cheio de jargões jurídicos e regras de relacionamento (como a tributação de batatas fritas 🍟).

Apenas após ler tudo (rolar o scroll até o final) e passar por um sistema de validação estressante, a verdadeira surpresa é revelada: o design corporativo some e dá lugar a um cartão romântico animado!

## ✨ Funcionalidades (O que tem sob o capô)

* **Validação de Leitura (Scroll):** O botão de aceite só é desbloqueado se o usuário rolar o texto até o último pixel. Se tentar trapacear e rolar para cima, o sistema bloqueia tudo de novo.
* **Gerenciamento de Estados:** Um modal dinâmico que muda de comportamento e "grita" com o usuário dependendo de quantas vezes ele clica no botão principal.
* **Troca de Temas Dinâmica:** Injeção de classes via JavaScript que altera todo o CSS da página instantaneamente.
* **Animações CSS Avançadas:** Uso de `@keyframes` para criar efeitos de surgimento e pulsação no cartão final.

## 🚀 Como rodar o projeto localmente

Não é necessário nenhum ambiente complexo para rodar este site.
